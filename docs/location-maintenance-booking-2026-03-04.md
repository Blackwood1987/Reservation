# 관리자 장소 유지보수 예약 기능 추가 (2026-03-04)

## 변경 전/후 대비
| 구분 | 변경 전 | 변경 후 | 영향 파일 |
|---|---|---|---|
| 장소 유지보수 예약 | 장소 목록 CRUD만 제공 | 관리자 전용 `장소 유지보수 예약` 모달 추가 | `app.html`, `app.js`, `styles.css` |
| 대상 선택 | 개별 장비 예약만 가능 | 현재 등록된 장소 전체 또는 일부 선택 가능 | `app.html`, `app.js` |
| 예약 생성 방식 | 사용자가 장비별로 수동 등록 | 선택 장소의 장비 전체에 동일 유지보수 예약 일괄 등록 | `app.js` |
| 충돌 처리 | 개별 등록 시점 확인 | 중복 시간대 장비는 자동 제외(건수 안내) | `app.js` |

## 동작 요약
- 관리자 탭 > 장소 관리에서 `+ 장소 유지보수 예약` 버튼으로 진입
- 날짜/시간/소요시간/작업명/사유 입력
- 전체 장소 또는 일부 장소 선택
- 저장 시 대상 장비별로 `purpose=maint`, `status=confirmed` 예약 생성

## 제약
- 관리자만 사용 가능
- 운영시간(09:00~18:00) 범위 내 등록
- 기존 예약과 시간 중복 시 해당 장비는 제외

## 검증
- `node --check app.js`
